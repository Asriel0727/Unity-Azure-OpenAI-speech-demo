## 專案簡介
在Unity中透過使用Microsoft Azure OpenAI Service功能以及Azure speech Server(TTS & STT)製作簡易對話機器人模板

## 專案功能
 - 自定義選擇對話語言(英文、繁體中文、日文)
 - 語音輸入對話內容
 - 語音輸出對話內容
